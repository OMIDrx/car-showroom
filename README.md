# Omid Motor Car Showroom

This project is a simple car showroom website including the following sections:

- Header: Website name and short slogan
- Features: 3 cards explaining car variety, fair pricing, and free consultation
- Car Slider: Featured car images with next/previous buttons
- Pricing Table: 3-row table showing model, year, mileage, and price
- FAQ: Simple accordion implemented with JavaScript
- Contact Form: Includes name, phone, message, and gender (radio buttons)

> This project is built with HTML, CSS, and JavaScript and is ideal for practicing frontend development.
